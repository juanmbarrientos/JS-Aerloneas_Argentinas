Proyecto en JavaScript - Aerolineas Argentinas

Aclaración: Este proyecto es de caracter personal y de práctica profesional. NADA tiene que ver la Empresa AEROLINEAS ARGENTINAS con la siguiente página.

Usando mayormente JavaScript, me propusé crear este proyecto de login, almacenamiento de datos en LocalStorage y SesionStorage combinados con un archivo JSON (simulando ser una API) que alimenta los datos visibles en el main y los devuelve en forma de Cards. Finalizada la estructura y funcionalidad de la web, centré mis habilidades UX y UI para reversionar parte del sitio de Aerolineas Argentinas. Este proceso fue una excelente manera de practicar y demostrar mi conocimiento aplicado a un Proyecto Integral de la vida real. A continuación detallo algunos conceptos con los cuales trabajé y su estado de desarrollo a la fecha ya que continuo agregando contenido.
